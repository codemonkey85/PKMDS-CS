# PKMDS-CS
[PKMDS C#](https://github.com/codemonkey85/PKMDS-CS "PKMDS C# on GitHub"): An open source Pokémon save hacking library for Generation VI (Pokémon X, Pokémon Y, Pokémon Omega Ruby, Pokémon Alpha Sapphire)
